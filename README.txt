1. SHORT DESCRIPTION

- This folder contains an assortment of programs and scripts to facilitate
scientific experiments and data analysis, written by Jinook Oh. [2024-08-10]


2. Dependencies for Python code

- Python 3.x

- Python package dependencies 
    - wxPython
    - openCV
    - sciPy
    - matPlotLib
    - tsmoothie
    - scikit-video
    - decord (optional)
    - pyueye (optional)
    - pyaudio (optional)

* All programs were written and tested on Ubuntu Linux 22.04 system.
